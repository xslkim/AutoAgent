## How to Run

```bash
# 1. Start the grounding model service (e.g., ShowUI-2B via vLLM)
python -m vllm.entrypoints.openai.api_server --model showlab/ShowUI-2B --port 8001

# 2. Capture real screenshots (or use placeholders)
#    Place notepad.png and calculator.png in tests/benchmarks/grounding/

# 3. Run the benchmark
python scripts/benchmarks/grounding_accuracy.py --endpoint http://localhost:8001/v1

# 4. Results are written to this file automatically
```

## Screenshot Capture

Real screenshots should be captured at **1920x1080** resolution with the target application
window maximized. Use the following procedure:

1. Open the target application (Notepad / Calculator)
2. Maximize the window
3. Take a screenshot and save as `notepad.png` or `calculator.png`
4. Place in `tests/benchmarks/grounding/`
5. Re-annotate bounding boxes in `targets.yaml` if needed
